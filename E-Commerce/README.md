1. Project Title
# E-Commerce Backend System


2. Project Description : 
A RESTful E-Commerce Backend application built using Java, Spring Boot, Spring Security, JWT Authentication, Hibernate, and MySQL.


3. Tech Stack : 

Java
Spring Boot
Spring Security
JWT
Hibernate
Spring Data JPA
MySQL
Maven
Postman
Git


4. Features : 

✔ User Registration & Login

✔ JWT Authentication

✔ Role-Based Authorization

✔ Product Management

✔ Category Management

✔ Shopping Cart

✔ Order Management

✔ Payment Module (Mock)

5. Project Structure : 

Controller
Service
Repository
Entity
Security
DTO
Configuration


7. Database Tables : 

Users

Products

Categories

Cart

Cart Items

Orders

Order Items

Payments


7. API Endpoints : 

POST   /api/auth/register

POST   /api/auth/login

GET    /api/products

POST   /api/cart/add/{id}

POST   /api/orders/checkout

POST   /api/payment/{id}

GET    /api/admin/dashboard

✔ Admin Dashboard

✔ BCrypt Password Encryption


8. How to Run : 

✔ Clone Repository

✔ Import into STS/Eclipse

✔ Create MySQL Database

✔ Update application.properties

✔ Run Spring Boot Application

✔ Test APIs using Postman
